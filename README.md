# Python_Jala_Assessment
# Python Assessment 1 & 2

This repository contains beginner-level Python programs completed as part of Assessment 1 and Assessment 2.

These programs help in understanding Python basics such as variables, loops, operators, conditions, and number programs.

---

# Assessment 1

## 1. Print Your Name
- Used the `print()` function to display output on the screen.

## 2. Single-line and Multi-line Comments
- Learned how to write comments in Python.
- `#` is used for single-line comments.
- Triple quotes `""" """` are used for multi-line comments.

## 3. Data Types Program
Learned different Python data types:
- Integer (`int`)
- Boolean (`bool`)
- Character/String (`str`)
- Float (`float`)

## 4. Local and Global Variables
- Understood variable scope in Python.
- Local variables work inside functions.
- Global variables can be accessed throughout the program.

# Assessment 2

## 1. Print “Bright IT Career” 10 Times
- Used `for` loop for repeated execution.

## 2. Print 1 to 20 Using while Loop
- Learned how `while` loop works.

## 3. Equal and Not Equal Operators
- Used comparison operators:
  - `==` Equal operator
  - `!=` Not equal operator

## 4. Print Odd and Even Numbers
- Used modulus operator `%` to check odd and even numbers.

## 5. Largest Number Among Three Numbers
- Used `if-elif-else` conditions to find the largest number.

## 6. Print Even Numbers Between 10 and 20
- Used `while` loop with conditions.

## 7. Print 1 to 10
- Learned looping concepts using `while` loop.

## 8. Armstrong Number Program
- Checked whether a number is an Armstrong number.

## 9. Prime Number Program
- Checked whether a number is prime or not.

## 10. Palindrome Program
- Checked whether a number reads the same forward and backward.

## 11. EVEN or ODD Using Switch
- Used Python `match-case` statement.

## 12. Gender Program Using Switch
- Printed gender based on user input (`M/F`).
